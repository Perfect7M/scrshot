# Scrshot

Package containing CLI for [Scrshot](https://scrshot.dev/).

## Installation

You can install them from npm package registry:

Using npm:

```bash
npm install @scrshot/cli
```

Using yarn:

```bash
yarn add @scrshot/cli
```

Using pnpm:

```bash
pnpm add @scrshot/cli
```
